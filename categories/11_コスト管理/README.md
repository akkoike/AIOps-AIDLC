# カテゴリ 11: コスト管理

## 概要
このカテゴリでは、インフラ業務のコスト管理・最適化を目的とした、以下の成果物を対象とします：

- **コスト分析**（固定/変動、部門別配分）
- **予算計画**（月次/年次）
- **削減施策**（ライセンス最適化、リソース右サイズ、使用停止）
- **コスト予測・フォーキャスト**
- **効果測定・KPI管理**

## SDD 7工程の実行順序（必須）

このカテゴリで取り扱うすべての依頼は、以下の工程順序を **厳密に守る** 必要があります：

1. **Specify**: What/Why を定義（削減対象、削減率目標など）
2. **Plan**: How を定義（分析方法、ツール選定）
3. **Tasks**: タスク分解（粒度: 30分以内）
4. **Implement**: コスト設計実装
5. **Verify**: テスト・検証実行
6. **Migration**: 本番展開手順
7. **Output**: 最終成果物

## 禁止事項

❌ ヒアリング直後の施策実装開始  
❌ requirements.md なしでの削減対象選定  
❌ 品質ゲート不合格のまま次工程へ進む
